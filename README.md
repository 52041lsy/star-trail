# BE MY AIS


### INTRODUCTION

**See the world together**
Be My Ais 利用人工智能技术，将让视障人士和低视力认识“看见世界”

### 主要功能
- 视障患者端
  - 智能“读屏”
    - 物体识别模式：识别镜头物体并播报
    - 文本识别模式：提取文本并朗读
    - 交通模式：识别交通灯、斑马线并通知
    - 景物模式：描述相机画面
  - 求助功能
    - 快速地向紧急联系人拨打电话求助
- 志愿者端
  - 获取绑定的视障患者位置信息
  - 快捷拨打电话查看情况
